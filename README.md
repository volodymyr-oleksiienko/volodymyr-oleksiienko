### 🚀 Main Pet-Project: Spec-Forge

**Live Production:** [v-oleksiienko.xyz](https://v-oleksiienko.xyz)

Spec-Forge is a fully functional, production-ready full-stack code generator that transforms JSON samples and schemas
into ready-to-use Java and TypeScript code. The system is split into 4 independent repositories, each with its own CI/CD
pipeline:

* **[Backend](https://github.com/volodymyr-oleksiienko/spec-forge-api):** Built with Java 25 & Spring Boot. Engineered
  with strict Hexagonal Architecture to isolate business logic.
* **[Frontend](https://github.com/volodymyr-oleksiienko/spec-forge-front):** Built with React 19, TypeScript, and Vite.
  Organized using Feature-Sliced Design (FSD) for high maintainability.
* **[Infrastructure](https://github.com/volodymyr-oleksiienko/spec-forge-infra):** Provisioned a DigitalOcean production
  environment using Infrastructure as Code (Terraform, Docker Compose, GitHub Actions).
* **[Plugin](https://github.com/volodymyr-oleksiienko/spec-forge-confluence-plugin):** Atlassian Forge configuration and
  CI/CD bridge for packaging and deploying the app to the Atlassian Marketplace.
